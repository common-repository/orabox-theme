The definitive skin for the Wordpress Admin Dashboard

== Description ==

This plugin allows users to customize the skin of their Wordpres Admin Dashboard in a couple of clicks.

How it works:

1. Install and Active
2. Enjoy it!

== Installation ==

Use WordPress' Add New Plugin feature, searching "Orabox Theme", or download the archive and:

1. Unzip the archive on your computer
2. Upload `Orabox-theme` directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu

== Frequently Asked Questions ==

= ¿Does this plugin make my website slower? =

Don't worry. Orabox Theme was created in order to give the possibility to change the Wordpress Admin Dashboard in a couple of clicks with the requirement of the lowest resources as possible. In other words, your website will be the same, nether slower or faster.

= ¿Do I need to upgrade my Wordpress to the latest version available?

It's always recommended to have your wordpress to the latest version, in order to keep your website safe and stable. On the other hand, you don't have to upgrade your wordpress to the latest version to use this plugin, if it's on at least the Wordpress 3.6
